Welcome to the Commento VS Code Extension!

Generate intelligent comments for your code using Google's Generative AI API.

What does Commento do?

Commento helps developers quickly generate comments for their code by leveraging the power of Google's Gemini AI. It supports multiple languages and provides concise or detailed comments based on the context of the code.

How to Use

  1. Install the Commento extension in your VS Code editor.
  2. Set up your API key in the settings by following these steps:
    • Open the Command Palette (use the shortcut Ctrl + Shift + P on Windows/Linux or Cmd + Shift + P on macOS).
    • Search for Preferences: Open Settings (JSON) and select it.
    • In the settings JSON file, add your API key as follows:
      •         {
            "commento.apiKey": "your-api-key-here"
        }
  3. To generate comments, follow these steps:
    • First, select the function or class in your editor that you want to generate comments for.
    • Open the Command Palette (use Ctrl + Shift + P or Cmd + Shift + P).
    • Type and select the command Commento: Concise Comment for concise comments or Commento: Detailed Comment for detailed comments.
  4. To manage your extensions and configure the Commento extension:
    • Open the Extensions view (Ctrl + Shift + X or Cmd + Shift + X).
    • Search for "Commento" in the Extensions Marketplace.
    • Click the gear icon next to the Commento extension and select Extension Settings to configure the API key or other preferences.
